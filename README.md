- **Payment**: Stripe
- **Authentication**: JSON Web Tokens (JWT)

## Getting Started

### Prerequisites
- Node.js (v14+)
- MongoDB
- Stripe Account

### Installation
1. Clone the repository
2. Install dependencies: `npm install`
3. Set up environment variables
4. Run the application: `npm start`
